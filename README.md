# TLCat - Termux 语言工具集

> 优雅轻量的 Termux 多语言环境管理工具🐱

## 特色

**开箱即用** - 无需复杂配置，安装即可开始使用  
**多语言支持** - 集成中文、日文等语言环境与实用工具  
**模块化设计** - 按需加载功能模块，保持系统清爽  
**极简体验** - 遵循现代设计理念，专注核心功能  

## 快速开始

### 安装
```bash
git clone https://github.com/FengTingXieShi/TLCat.git ~/TLCat
cd ~/TLCat
./install.sh
```

使用

安装完成后，重新启动 Termux 或执行：

```bash
source ~/TLCat/bin/init-tlc
```

现在就可以在任意目录使用 TLCat 提供的各种命令啦～ 喵～
